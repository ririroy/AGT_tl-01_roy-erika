Section A

Alpha

La valeur alpha sert à changer la transparence d'un élément.

Exemples de transparence avec alpha:

RGBA: rgba(255, 0, 0, 0.5)
HSLA: hsla(120, 100%, 50%, 0.3)
Opacity: 0.60

Section D

Display Flex

Flex permet d'aligner les éléments d'un conteneur. Par exemple, justify-content aligne les éléments sur l'axe des X, alors que align-items le fait sur l'axe des Y. Il y a aussi flex-direction qui est souvent utilisé et qui permet de mettre les éléments en column ou en row.

.exemple-d {
    display: flex;
    flex-direction: column;
    gap: 1rem;
}

Section F

Filter

Avec filter, il est possible d'ajouter des filtres à des images.

.img-f1 {
    filter: hue-rotate(100deg);
}

.img-f2 {
    filter: blur(3px);
}
Section G

Grid

Avec grid, on peut transformer un conteneur en grille et choisir le nombre de colonnes et de rangées avec grid-template-columns et grid-template-rows.

.exemple-g {
    gap: 1rem;
    display: grid;
    grid-template-columns: 1fr 1fr 1fr;
}
Section H

Hover effects

On peut styliser un élément au survol avec :hover.

Voici un exemple:

.survolerBtn {
    cursor: pointer;
    border: none;
    font-size: 1rem;
    padding: 15px 32px;
    margin: 4px 2px;
    background-color: #af4c8c;
    color: white;
    border-radius: 15px;
}

.survolerBtn:hover {
    background-color: #62244c;
    box-shadow: 0 5px 15px rgba(0, 0, 0, 0.2);
}

Section K

@keyframes

La règle CSS @keyframes permet la création d'animations directement dans CSS, celle-ci étant utilisée avec au moins une propriété d'animation. Il est possible de styliser et de choisir la vitesse de l'animation.

Voici un exemple d'animation avec @keyframes:

.exemple-k {
    width: 50px;
    height: 50px;
    background-color: hotpink;
    animation: color-move 3s infinite;
}

@keyframes color-move {
    0% {
        background: purple;
        transform: translateX(0);
    }
    50% {
        background: hotpink;
        transform: translateX(100px);
    }
    100% {
        background: purple;
        transform: translateX(0);
    }
}
Section L

Linear-gradient

Avec linear-gradient, il est possible de créer des dégradés linéaires dans des éléments en CSS.

.exemple-L {
    width: 50px;
    height: 50px;
    background: linear-gradient(rgb(113, 9, 67), rgb(255, 58, 189));
}
Section M

Media queries

Les Media Queries permettent d'ajuster l'affichage de la page selon les différentes tailles d'écran.

Voici un exemple de la façon dont on les utilise:

@media screen and (min-width: 768px) {
    h1 {
        font-size: 25px;
    }
    nav {
        display: flex;
        flex-wrap: wrap;
        justify-content: center;
    }
    nav a {
        margin: 0 12px;
        font-size: 17px;
        }
}

Section P

Perspective

Avec la propriété perspective, on peut donner de la profondeur en 3D aux éléments enfants d'un conteneur, créant ainsi un effet de perspective.

.front { transform: translateZ(50px); }
.back { transform: rotateY(180deg) translateZ(50px); }
.right { transform: rotateY(90deg) translateZ(50px); }
.left { transform: rotateY(-90deg) translateZ(50px); }
.top { transform: rotateX(90deg) translateZ(50px); }
.bottom { transform: rotateX(-90deg) translateZ(50px); }

@keyframes rotateCube {
    0% { transform: rotateX(0) rotateY(0); }
    100% { transform: rotateX(360deg) rotateY(360deg); }
}

Transform

La propriété transform permet d'appliquer des transformations sur un élément 2D ou 3D, dont la rotation, la translation, etc.

Voici un exemple:

.forme1 {
    transform: rotate(0.2turn);
    background-color: orange;
}

.forme2 {
    transform: skew(15deg, 20deg);
    ackground-color: green;
}

.forme3 {
    background-color: red;
    transform: scale(1, 0.5);
}

.forme4 {
    background-color: blue;
    transform: translate(1rem, 1rem);
}

.perspective {
    perspective: 200px;
}

.forme5 {
    background-color: hotpink;
    transform: rotateX(30deg);
}

.forme6 {
    background-color: purple;
    transform: rotateY(30deg);
}
